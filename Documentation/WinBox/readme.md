### IP Tables
```
Prerouting  : Paquet entrant dans le routeur.
Forward     : Paquet qui traverse le routeur.
Postrouting : Paquet sortant du routeur.
```

### QoS
```
Prerouting = Envoie
Forward    = Téléchargement
```


### Maitrise
```
On peux manipuler que sur le flux entrant et pas sortant !
```
